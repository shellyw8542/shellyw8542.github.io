SEP 12 Freedom Project (endless enemies)

Context

For my freedom project, I wanted to create another interactive game using Godot. Godot is a very powerful game engine that allows for both 2D and 3D game development. Since there are a lot of popular games that use Godot, I decided to explore its capabilities.

Content

My project is mainly about attacking entities using animation features, a score counter, HP, and attack mechanics in Godot. My partner and I developed collaboration and communication skills while learning this game engine. We also relied on watching tutorials and asking for help when needed to better understand how to implement different features.

Reflection

My biggest takeaway from this project was how important communication and time management really are. Managing my time was honestly one of the biggest challenges, since I had to balance studying for the APCSA exam while also working on the Freedom Project. I still wanted to stay passionate about what I was building and make something that felt worth the time and effort, and actually fun to play. In the end, my partner and I are really proud of what we created, especially since we earned an Honorable Mention.
